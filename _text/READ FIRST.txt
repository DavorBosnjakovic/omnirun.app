# Omnirun -- Project Knowledge Overview
# Updated: March 26, 2026

## Product Name: Omnirun
## Domain: omnirun.app
## Tagline: "Run everything. Describe it. Done."

## What is Omnirun?
A desktop application (Tauri) that lets anyone build software, automate tasks,
and control their computer by describing what they want -- typing or voice.
Not just a dev tool -- it runs anything. Now expanding into a proactive personal
agent platform with email, calendar, and watching agent capabilities.

## Key Technical Decisions
- Desktop app: Tauri (Rust + React + TypeScript)
- BYOK model (users bring own Anthropic API key)
- Direct distribution (no app stores)
- SQLite for local data
- Supabase for cloud backend (auth, billing, team, analytics)

## App Navigation Structure (as of March 26, 2026)
Four top-level sidebar sections:
1. Home -- dashboard (Morning Brief, widgets for projects/email/tasks/usage/connections)
2. Projects -- existing build/chat experience, unchanged
3. Assistant -- global chat for personal AI (email agents, calendar, watching agents)
4. Tasks -- Scheduled Tasks view (was Tools dropdown only, now top-level)

## Two Types of Connections (CRITICAL DISTINCTION)
- Connections (Settings page) = project services: Vercel, GitHub, Stripe, Supabase etc.
- Assistant connections = personal integrations: Gmail, Outlook, Calendar etc.
These are NEVER mixed. Email connections live in the Assistant section, not Settings.
